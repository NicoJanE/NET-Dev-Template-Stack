## What does this directory contain

- `Create-API-Documentation.md` Instruction for creating API documentation in the code and for generating DocFX browser documentation of it,
- `.\Template\page.md` DocFX supports markdown, frontmatter and it some DocFX specific syntax command, those are brief summarized in this document

More information: [DocFX website](https://dotnet.github.io/docfx/)
